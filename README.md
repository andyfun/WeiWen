这是一个干货网和界面网的综合新闻客户端，可访问干货集中营Android、拓展资源、瞎推荐以及界面网科技模块的新闻，   
图片框架用的Picasso，网络框架用的Volley和Retrofit2+okhttp，实现了缓存技术，适合练习框架的使用。
